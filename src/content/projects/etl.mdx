# Implementación de Data Warehouse y ETL

Este proyecto se centra en la implementación de un **Data Warehouse** con procesos **ETL (Extracción, Transformación y Carga)** para optimizar la toma de decisiones empresariales. Se diseñó una estructura de almacenamiento de datos consolidada, transformando información de bases de datos **OLTP** en un entorno **OLAP**, facilitando el análisis y generación de reportes.

## Mi Rol en el Proyecto

Fui el **coordinador del equipo de desarrollo**, encargado de organizar y supervisar las actividades para garantizar el cumplimiento de los objetivos del proyecto. Mis responsabilidades incluyeron:

### Gestión y Planificación
- Coordiné la distribución de tareas entre los miembros del equipo, asegurando un flujo de trabajo eficiente.
- Supervisé la implementación de cada fase del proceso **ETL**, asegurando la correcta integración entre la base de datos OLTP y el Data Warehouse.
- Organicé reuniones periódicas para evaluar avances y resolver bloqueos técnicos, asegurando que el proyecto cumpliera con los requisitos establecidos.

### Diseño e Implementación del Data Warehouse**
- Definí la arquitectura de la base de datos OLAP, estableciendo las **dimensiones, métricas y tablas de hechos** para optimizar la analítica empresarial.
- Supervisé el diseño de **consultas SQL optimizadas**, garantizando la eficiencia en la extracción de datos desde la base OLTP.

### Desarrollo del Proceso ETL
- Dirigí la implementación del proceso ETL utilizando **Java y SQL**, asegurando la transformación y carga de datos con calidad y precisión.
- Coordiné la validación de los datos extraídos, implementando mecanismos de detección y corrección de errores en la carga de información.
- Supervisé la integración con herramientas de análisis de datos como **Tableau y Pentaho**, facilitando la generación de reportes interactivos.

### Optimización y Pruebas
- Realicé pruebas de rendimiento para garantizar tiempos de respuesta óptimos en las consultas de análisis.
- Supervisé la seguridad y consistencia de los datos, asegurando la confiabilidad de la información almacenada en el Data Warehouse.
- Coordiné la documentación del sistema, facilitando futuras mejoras y mantenimiento.

Este rol me permitió fortalecer mis habilidades en **gestión de proyectos, liderazgo técnico y modelado de bases de datos**, asegurando el éxito del desarrollo e implementación del Data Warehouse.

## Tecnologías Utilizadas

- **Lenguaje de Programación**: Java, SQL.
- **Bases de Datos**: MySQL (OLTP), PostgreSQL (OLAP).
- **ETL**: Java, SQL, Pentaho Data Integration.
- **Herramientas de BI**: Tableau.
- **Gestión del Proyecto**: GitHub, Scrum.

## Implementación

Este proyecto fue desarrollado en el curso **IS-601 Bases de Datos II** en la **Universidad Nacional Autónoma de Honduras (UNAH)**. Su propósito fue demostrar la aplicación real de un **Data Warehouse y ETL**, optimizando la gestión de datos empresariales para la toma de decisiones estratégicas.
